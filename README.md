# AgriHub 

AgriHub is a modern, AI-driven agriculture platform focused on Indian farming ecosystems.  
It combines clean UI, multilingual access, and cloud-backed scalability to deliver practical agricultural insights.

## What It Does
- Provides agriculture-focused digital solutions
- Supports multiple Indian languages
- Designed for accessibility and real-world usage
- Built with a scalable, production-ready architecture

## Tech Overview
- Frontend: Modern UI with smooth transitions and responsive design
- Backend & Database: Supabase
- Focus Areas: Indian agriculture, usability, scalability
